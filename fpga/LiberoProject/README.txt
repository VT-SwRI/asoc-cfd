The folder named "FullSystem" contains the entire Libero project structure. To open the project, ensure Libero v11.9 is installed on your machine and open the file named "FullSystem.prjx".

YOU MUST INSTANTIATE A MULTIPLIER BLOCK FROM THE LIBERO CATALOG FOR THE PROJECT TO COMPILE CORRECTLY. Refer to the project binder, or the detailed design document for a guide on how to instantiate the multiplier correctly.

All HDL source code, GUI source code, and additional scripts can be found in the source code file.
